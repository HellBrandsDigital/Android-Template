# Template for an Android App

This is a Template for a new Android App.

You need to set the correct Sonar Properties in build.gradle for it to work.

It includes:
- Kover for Code Coverage
- SonarQube
- Robolectrics for Unit-Testing
- A raw Structure & sample unit tests

This took quite some time to find a working Solution for Sonar with Code Coverage.
JaCoCo didn't worked at all.
